Installing React dependencies:
  1. Open terminal
  3. Run "cd client"
  2. Run "npm install"
Start flask server:
  1. Open another terminal
  2. Run "cd flask-server"
  3. Run "source venv/bin/activate"
  4. Run "python3 server.py"
Start React:
  1. Go back to terminal where you installed React dependencies
  2. Run "npm start"
